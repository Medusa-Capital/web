## Project Aesthetic

Neo-Noir / Cyber-Luxury: dark backgrounds, purple/blue gradient accents, premium typography, institutional authority. Crypto-native but compliance-first.

## UI Type Defaults

- Landing / homepage: marketing landing (use `patterns/landing-page.md`)
- Lead capture / forms: forms pattern (use `patterns/forms.md`)
- Navigation: top nav sticky (use `patterns/navigation.md`)
- Auth (members area): minimal dark auth (use `patterns/auth.md`)
